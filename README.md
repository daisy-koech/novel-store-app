# Novel Haven (Personal Project Showcase)
- Novel Haven is a React single-page application that functions as an online bookstore for novels. 
- Users can view novels, search dynamically, and administrators can add new novels. 
- This project implements React hooks, state management, client-side routing, and data fetching from a simulated backend.

## Features
- Landing Page: Displays store information (name, description, and contact).
- Novels Page: Shows all novels with cover images, author, genre, and price.
- Add Novel Page: A form for administrators to add new novels.
- Search Functionality: Dynamic search by title, author or genre.

## Technologies used:
-React (components & hooks)
-React Router DOM
-json-server (simulated backend)
-CSS

## Installation
- Clone the repository
- Install dependencies
- Start JSON Server
- Start React App using 'npm run dev'
- Live site is hosted on vercel
